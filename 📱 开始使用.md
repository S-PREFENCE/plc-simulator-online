## 🔧 常用操作

### 添加元件
1. 点击指令面板中的指令（会高亮）
2. 点击梯级中的 **+** 按钮
3. 元件自动添加到当前梯级

### 编辑属性
1. 点击已添加的元件
2. 修改地址（如 I0.0 → I0.1）
3. 修改预设值（定时器等）
4. 点击"确定"保存

### 删除元件
1. 点击元件
2. 在弹出窗口中点击"删除"

### 测试程序
1. 在底部变量监视器
2. 点击输入变量（I0.0 等）切换状态
3. 观察输出变量（Q0.0 等）变化
4. 点击"运行"执行逻辑

## 💡 示例程序

### 简单启保停电路

```
梯级 1:
|  I0.0  ──┬── I0.1  ── Q0.0  ──|
|  Q0.0  ──┘
```

**说明：**
- I0.0: 启动按钮（常开）
- I0.1: 停止按钮（常闭）
- Q0.0: 电机输出
- Q0.0 自锁触点

**操作：**
1. 添加 I0.0（常开触点）
2. 添加 I0.1（常闭触点）
3. 添加 Q0.0（线圈）
4. 再添加一个 Q0.0（常开触点）与 I0.0 并联
5. 测试：切换 I0.0，观察 Q0.0

## 🐛 故障排除

### 手机无法访问
1. 检查 Wi-Fi 连接（必须同一网络）
2. 确认 IP 地址正确
3. 关闭防火墙测试
4. 尝试其他端口

### 页面显示异常
1. 清除浏览器缓存
2. 刷新页面
3. 更换浏览器（推荐 Chrome/Safari）

### 元件无法添加
1. 确保已选择指令（会高亮）
2. 点击梯级中的 **+** 按钮
3. 检查浏览器控制台错误
